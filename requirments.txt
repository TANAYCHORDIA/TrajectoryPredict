TrajectoryPredict Environment Notes
=================================

Primary environment file
- Use `env.yml` for reproducible setup with Conda.

Create the environment
- conda env create -f env.yml
- conda activate trajpredict

Update after dependency changes
- conda env update -f env.yml --prune

Quick verification
- python -c "import torch; print(torch.__version__)"
- python -c "import nuscenes, wandb, matplotlib, numpy; print('imports ok')"

What this includes
- Python 3.10
- PyTorch 2.1
- CUDA runtime via `pytorch-cuda=11.8`
- Core Python packages: nuscenes-devkit, wandb, matplotlib, numpy

Notes
- Keep dependency changes synchronized between `env.yml` and project docs.
- For hackathon speed, prefer this single environment setup path for all team members.
